Usage

Step 1: Ajax in the SVG file from our CDN

Use this snippet (vanilla JS) to AJAX our composite SVG sprite that is hosted on our Amazon S3 CDN. This should be gzipped and will be approx >~150kb. 

<script>
var ajax = new XMLHttpRequest();
ajax.open("GET", "https://assets.gi.org/icons/icons.svg", true);
ajax.send();
ajax.onload = function(e) {
  var div = document.createElement("div");
  div.innerHTML = ajax.responseText;
  document.body.insertBefore(div, document.body.childNodes[0]);
}
</script>

Step 2: Include the inline SVG/USE code

Add a class of .icon to the SVG tag, add an additional class to SVG that corresponds to the specific icon (see below), and reference the same exact corresponding hash URL (which is a marker in the SVG file that serves as an "anchor" for the icon in question).

Example
<svg class="icon icon-home2"><use xlink:href="#icon-home2"></use></svg>
