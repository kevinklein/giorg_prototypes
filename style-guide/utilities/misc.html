Display

.display-block

Block display for any element, all contexts.

.display-inline-block

Inline-lock display for any element, all contexts.

.display-inline-flex

Usually when things are display:flex by default, you sometimes need that to be an inline flex element.

.display-sm-block, .display-lg-block

Block display for small and large screens, respectively.

.is-hidden

Hide something by default (i.e. for toggling via Javascript)

Overflow

.overflow-hidden

Hide overflow on any element.

.overflow-auto

Scroll overflow on any element.

Border Utilities

.border-all, .border-top, .border-right, .border-bottom, .border-left

Adds a slight, gray border (black with opacity) to any element on any specific vector.

.border-inverse

Inverse the border to white with opacity.

.border-none, .border-top-none, .border-right-none, .border-bottom-none, .border-left-none

Granularly remove borders.

Other

.cursor-pointer

Turns mouse arrow into a cursor when hovering an element.
