HTML containers are pretty simple in that they are 100% by default and scale up with minimum widths depending on browser width. They can be nested inside of stripes, other container divs, or another element to enforce max widths. General, grid items are nested in-between.

.container

Max width 72.25rem (1156px).

.container-full

100% width, no maximum widths. Padding on the left/right as a default.

.container-lg

Max width 90rem (1440px).

.container-sm

Max width 48rem (768px).

.container-xs

Max width 36rem (576px).
