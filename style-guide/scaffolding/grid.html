Use the following classes to proportionately lay grid scaffolding units into a layout. Simply wrap the individual columns in a .row, for instance: 

<div class="row">
    <div class="col-xs-8">I am a 2/3 width column</div>
    <div class="col-xs-4">I am a 1/3 width column</div>
</div>

Responsive

The example below will create grid items for larger (desktop and tablet) screens, with stackable divs on smaller screens. To force grids onto small (in this case what is generally understood to be mobile) screens, use .col-xs-#.

.col-md-6.col-xs-12
.col-md-6.col-xs-12
<div class="row">
    <div class="col-md-6 col-xs-12">1/2 width column that breaks down to a full column in mobile</div>
    <div class="col-md-6 col-xs-12">1/2 width column that breaks down to a full column in mobile</div>
</div>
