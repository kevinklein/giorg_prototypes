De-emphasized text

.text-mute: Fusce dapibus, tellus ac cursus commodo, tortor mauris nibh.

.text-muted: Fusce dapibus, tellus ac cursus commodo, tortor mauris nibh.

.text-opaque-dark: Has a low opacity, and best used against light backgrounds.

.text-opaque-light: Has a low opacity, and best used against dark backgrounds.

Theme colored text

You can use any of the colors, with a prefix of .text- (and a lowercasing of the color name, subbing a dash for spaces), i.e.:

.text-info: Maecenas sed diam eget risus varius blandit sit amet non magna.

.text-primary: Nullam id dolor id nibh ultricies vehicula ut id elit.

.text-secondary: Nullam id dolor id nibh ultricies vehicula ut id elit.

.text-secondary-dark: Nullam id dolor id nibh ultricies vehicula ut id elit.

.text-orange: Maecenas sed diam eget risus varius blandit sit amet non magna.

.text-orange-dark: Maecenas sed diam eget risus varius blandit sit amet non magna.

.text-orange-light: Maecenas sed diam eget risus varius blandit sit amet non magna.

.text-warning: Etiam porta sem malesuada magna mollis euismod.

.text-danger: Donec ullamcorper nulla non metus auctor fringilla.

.text-success: Duis mollis, est non commodo luctus, nisi erat porttitor ligula.

.text-dark: Maecenas sed diam eget risus varius blandit sit amet non magna.

.text-black: Maecenas sed diam eget risus varius blandit sit amet non magna.

Inverse Text

Add a class of .text-inverse to an element or a parent and the element (or its children) will become white.
